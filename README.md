# Simulate WhatsApp Third Party AI Assistant

This example simulates a basic third-party AI assistant, similar to those described for WhatsApp integration. It processes user input and provides rule-based responses for tasks like summarization, translation, and general information, demonstrating the core concept of an external AI service interacting with user messages.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
Type messages and observe the assistant's responses.
